### Desenvolvendo
Para começar o desenvolvimento, primeiro verifique se o Node está instalado e atualizado, depois instale as dependências e comece a desenvolver!

```bash
# Instalando as dependências.
npm install

# Iniciando o Vite.
npm run dev
```
